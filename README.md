# Green Cart Ltd - Sales & Customer Behaviour Insights

## Project Overview
This project analyzes sales performance and customer behaviour for Green Cart Ltd, a UK-based e-commerce company specializing in eco-friendly household products.

The objective was to clean, merge, and analyze sales, product, and customer datasets to identify revenue trends, customer purchasing patterns, product performance, and delivery efficiency.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Tasks
- Data Cleaning
- Data Merging
- Feature Engineering
- Exploratory Data Analysis
- Business Insights
- Data Visualization

## Features Created
- Revenue
- Order Week
- Price Band
- Days to Order
- Email Domain
- Late Delivery Flag

## Visualizations
- Weekly Revenue Trends by Region
- Top Product Categories by Revenue
- Quantity vs Discount Analysis
- Correlation Heatmap
- Orders by Loyalty Tier
- Delivery Status by Price Band

## Key Business Questions Answered
- Which product categories generate the most revenue?
- Do discounts increase sales volume?
- Which loyalty tier generates the most value?
- Which regions experience the most delivery delays?
- How do customer signup patterns affect purchasing behaviour?
